More Information About This Project

Libraries Used

  • PyTorch: For building and training the logistic regression model.
  • NumPy: For numerical computations and data manipulation.
  • Scikit-learn: For text preprocessing and feature extraction using CountVectorizer and TfidfTransformer.
  • Pandas: For dataset handling and preparation.

Logistic Regression Model

The spam detector relies on a custom logistic regression model implemented from scratch using PyTorch. Key details:

  • Model Type: Binary classification (Spam vs. Not Spam).
  • Feature Extraction: Text data is transformed into numerical features using CountVectorizer and TfidfTransformer.
  • Training Dataset: The model is trained on the Kaggle SMS Spam Collection Dataset, containing labeled spam and ham messages.
  • Implementation: The logistic regression model is built and trained entirely in PyTorch, with no pre-trained models used.
  • Optimization: The model uses the Binary Cross-Entropy Loss function (BCELoss) and is optimized using the torch.optim.Adam optimizer with a learning rate of 0.001.
  • Accuracy of Model: ~ 98.08%

Model Workflow

  1. Text Preprocessing: Text data is cleaned by lowercasing, removing stopwords, and tokenization.
  2. Feature Extraction: The processed text is transformed into a numerical matrix using TfidfTransformer.
  3. Model Definition: A logistic regression model is implemented in PyTorch with a single linear layer.
  4. Model Training: The model is trained on the preprocessed and vectorized training data using gradient descent.
  5. Prediction: Input messages are classified as Spam or Not Spam using the trained PyTorch model.

Future Implementation

In future versions of the project, a machine learning model will be incorporated that learns from user data, helping the system adapt and improve its predictions based on diverse data for better performance. This dynamic learning approach will allow the system to continually refine itself and provide more accurate and context-aware classifications, contributing to a more efficient and personalized spam detection system.
